Sobre o Visualizador do PowerPoint

O Visualizador do Microsoft Office PowerPoint possibilita a exibição de apresentações criadas no Microsoft Office PowerPoint 2000, PowerPoint XP, PowerPoint 2003 e PowerPoint 2007. O Visualizador do PowerPoint também oferece suporte à abertura de apresentações do PowerPoint protegidas por senhas.

OBSERVAÇÃO: para exibir apresentações criadas no PowerPoint 2007, é necessário instalar o Pacote de Compatibilidade para Formatos de Arquivo do Microsoft Office Word, Excel e PowerPoint 2007. Consulte http://office.microsoft.com/ para obter mais informações.

Requisitos do sistema

  • PC com processador de 200 MHz ou superior
  • Microsoft Internet Explorer 5 ou posterior
  • Sistemas operacionais com suporte: Microsoft Windows 2000 com Service Pack 4, Windows XP com Service Pack 1, Windows Server 2003 ou Windows Vista.
  • Memória RAM:
    • Windows 2000 com Service Pack 4: 64 MB de RAM
    • Windows XP com Service Pack 1, Windows Server 2003: 128 MB de RAM
    • Windows Vista: 512 MB de RAM
  • 5 MB de espaço disponível em disco rígido
  • Super VGA (800 x 600) ou resolução de tela superior

Problemas relacionados a idiomas

O Visualizador do PowerPoint não oferece suporte aos Revisores de Texto do Microsoft Office.

O Visualizador do PowerPoint não oferece suporte a apresentações em idiomas bidirecionais criadas no PowerPoint 97.

Trabalhando com o Visualizador do PowerPoint

O Visualizador do PowerPoint não mostra animações de diagramas corretamente. Em vez de mostrar as animações, o diagrama é exibido concluído.

O Visualizador do PowerPoint não oferece suporte à abertura de apresentações do Gerenciamento de Direitos de Informação (IRM).

O Visualizador do PowerPoint não oferece suporte à execução de macros, programas ou abertura de objetos vinculados ou incorporados.

O Visualizador do PowerPoint não oferece suporte à exibição destes formatos gráficos quando vinculados a uma apresentação: .eps, .pct, .emz, .cdr, .cgm, .wpg e .wmz.
